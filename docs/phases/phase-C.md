# Phase C — Campaign days

**Unlock:** Phase B acceptance criteria met. **Unlocked.**  
**Note:** Required before Phase G. May be deferred past Phase D if you prioritise carry/parts first (see sequencing exceptions).

## Goal

Linear **days** (Day 1, Day 2, …). Each day is one shift (plus retries). Rising spawn pressure. Fail a day → retry that day. Pass/retry uses Phase B’s % gate; copy can use “don’t come back tomorrow” flavour. This is **not** Phase G sacking.

## In scope

- Day index + save/continue as needed
- Per-day spawn/timer parameters
- Progress to next day only on pass
- Terminology: campaign **day** wraps a **shift**

## Out of scope

- Between-day shop (Phase F)
- Campaign-ending reputation sacking (Phase G)

## Acceptance criteria

- [ ] Completing a day unlocks the next
- [ ] Failing keeps you on the same day
- [ ] Difficulty trends upward across early days
- [ ] UI distinguishes day number from shift timer
