# Phase G — Reputation & sacking

## Goal

Anger/reputation accumulates across days. Too many failures → sacked (true game over), evaluated in the “don’t come back tomorrow” moment — not a random mid-shift kill unless design later wants a warning spike.

## In scope

- Reputation meter
- Cross-day accumulation rules
- Game over / sacked screen
- Warnings before the final sack

## Out of scope

- Permadeath roguelike meta beyond restart campaign

## Acceptance criteria

- [ ] Persistent reputation across days
- [ ] Crossing threshold ends the run with a clear sack message
- [ ] Player can see they were in danger before it happens
