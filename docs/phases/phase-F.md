# Phase F — Stock & budget

**Unlock:** Phase D complete (parts exist to stock). **Phase E optional** — economy may ship against the same-map closet only.  
**Typically after Phase C** so “between days” has a campaign to attach to; if C is deferred, between-shift shop is an acceptable interim.

## Goal

Between days (or between shifts if C is deferred), manage IT budget and order peripherals (toner, etc.). Shortages can block matching fixes. Leftover “waiting on order” can carry into the next morning.

## In scope

- Between-day (or between-shift) shop / order UI
- Budget currency from performance
- Stock inventory affecting ticket solvability
- Optional: next shift starts with leftover blocked tickets

## Out of scope

- Full accounting sim depth
- Campaign-ending sacking (Phase G)

## Acceptance criteria

- [ ] Player can spend budget on at least two item types
- [ ] Zero stock blocks the matching fix until ordered
- [ ] Orders arrive on a clear rule (e.g. next day/shift)
